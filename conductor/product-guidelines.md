# 1. 散文风格

所有文档，包括技术文档、用户手册和报告，都必须遵循正式和技术化的叙述风格。内容应精确、客观，并使用行业标准术语。

# 2. 视觉识别

应用的视觉设计应体现专业性和极简主义。

*   **色彩方案**：以中性色调为主，如蓝色、灰色、白色，以确保界面整洁、不分散注意力。强调色应谨慎使用，仅用于高亮关键操作或信息。
*   **界面**：设计应注重功能性，避免不必要的装饰元素。布局应清晰、一致，并针对高效的工作流程进行优化。

# 3. 对外沟通

所有对外沟通，包括发布说明、用户邮件和博客文章，都应采用权威和信息丰富的语气。

*   **重点**：沟通内容应强调我们的专业知识和对细节的关注，旨在建立用户信任并提供深入见解。
*   **风格**：语言应清晰、精确，避免使用模棱两可或过于随意的措辞。

# 4. 市场定位

产品应被定位为一个可靠、创新且用户友好的解决方案，以吸引广泛的用户群体。

*   **可靠的效率工具**：在所有营销和产品信息中，强调应用的稳定性、卓越性能和对工作流程的优化。
*   **创新的技术先锋**：突出独特的技术解决方案和前瞻性功能，以吸引早期采用者和技术爱好者。
*   **用户友好的解决方案**：侧重于易用性、直观界面和出色的用户体验，降低学习曲线。
