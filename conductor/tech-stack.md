# 1. 核心技术

*   **编程语言**:
    *   **TypeScript**: 用于前端开发，提供静态类型检查。
    *   **Rust**: 用于后端开发，确保性能和内存安全。
*   **前端框架**:
    *   **Vue.js**: 一个渐进式的JavaScript框架，用于构建用户界面。
*   **桌面应用框架**:
    *   **Tauri**: 一个用于构建跨平台桌面应用的框架，使用Web技术作为前端。

# 2. 构建与样式

*   **构建工具**:
    *   **Vite**: 一个现代化的前端构建工具，提供极速的开发服务器和优化的构建输出。
*   **CSS框架**:
    *   **Tailwind CSS**: 一个功能类优先的CSS框架，用于快速构建自定义设计。

# 3. 数据层

*   **ORM**:
    *   **SeaORM**: 一个用于Rust的异步和动态ORM。
*   **数据库**:
    *   **关系型数据库**: 具体类型待定，但项目使用SeaORM进行数据交互。
